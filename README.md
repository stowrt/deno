# deno-mongo-rest-api

REST api in deno using mongo

## Steps:

1. Clone the repo
2. Run command:
   - \$ deno run --unstable --allow-read --allow-write --allow-net --allow-plugin main.ts

## Library Used:

> 1. mongo - plugin for connecting deno with mongodb.
> 2. oak - middleware library like express, but for deno.
